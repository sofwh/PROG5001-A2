PROJECT TITLE: Student Statistics Management System

PURPOSE OF PROJECT:
The Student Statistics Management System is a Java program designed to read and analyze student data from a text file. It provides various functionalities, including calculating and printing total marks, identifying students below a threshold, and displaying the top-performing and bottom-performing students.

VERSION or DATE: Version 1.0 (September 2023)

HOW TO START THIS PROJECT:
1. Clone the repository to your local machine.
2. Ensure you have Java installed (version 8 or higher).
3. Open the project folder in your preferred Java development environment .
4. Compile and run the "StudentStatistics.java" file.

AUTHORS:
- Sirjan Baniya

USER INSTRUCTIONS:
1. When you run the program, you will be prompted to enter the name of the input file containing student data. Please provide the full path to the file.

2. The program will display a menu with the following options:
   - Calculate and Print Total Marks
   - Print Students Below a Threshold
   - Print Top 5 Students with Highest Total Marks
   - Print Top 5 Students with Lowest Total Marks
   - Exit

3. Select the desired option by entering the corresponding number.
   - Option 1: Calculates and prints the total marks of all students.
   - Option 2: Enter a threshold, and the program will print students with total marks below that threshold.
   - Option 3: Prints the top 5 students with the highest total marks.
   - Option 4: Prints the top 5 students with the lowest total marks.
   - Option 5: Exits the program.

4. Follow the on-screen instructions to interact with the program and retrieve the desired statistics.

5. The program handles various scenarios, including reading data from the input file, validating input, and displaying results.

6. Enjoy using the Student Statistics Management System for analyzing student performance data!
